# How to install PSoC designer V5.0 under Windows 10
What worked for me ...
1. Install SVG Viewer (zip file link is [here on Infineon Community Forum](https://community.infineon.com/t5/PSoC-5-3-1/Adobe-SVG-viewer-not-available-and-PD-5-4-SP1-won-t-work/m-p/77237#M10104))
2. Install [PSoCProgrammerSetup_3.24.2.b2254_RS](https://itools.infineon.com/archive/PSoCProgrammerSetup_3.24.2.b2254_RS.exe)
3. Install [PSoC_Designer_5.0_Service_Pack_6](https://itools.infineon.com/archive/PSoC_Designer_5.0_Service_Pack_6.zip)
